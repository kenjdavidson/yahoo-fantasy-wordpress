Yahoo Fantasy Sports provdies a the plugin [yahoofantasysports], which inserts
specific Yahoo Fantasy content into your posts and pages.

== Description ==

If you're looking to add Fantasy Sports content into your posts, then this plugin
is for you.  Once configured, the plugin allows you to insert team, league, game,
standing, etc. information into your posts and pages uses a single shortcode.

== Installation ==

To install the Yahoo Fantasy Sports plugin:

1) Download the plugin zip
2) Extract / Upload to your Wordpress plugin folder
3) Complete the Yahoo Developer registration
4) Complete the Yahoo OAuth Configuration
5) Edit pages and posts
