Yahoo! OAuth Keys

In order for this Wordpress site to allow users to include Yahoo! Fantasy Sports details on their posts and pages, you must (as an administrator) register your page on the Yahoo! developer network. In order to do this, follow these steps:

  1. Open the Yahoo! Developer Network portal and sign in.
  2. Hover over the My Apps menu in the top right corner, or go to the My Apps page.
  3. Click the Create App button.
  4. Complete the new application form:
    • Application Name - The friendly name of your App/Wordpress page
    • Application Type - Choose Web Application
    • Description - Optional
    • API Permissions - Select Fantasy Sports (read). Currently there is only read functionality, but in the future if any more options are changed, a new Application will need to be created

Once completed you'll be provided with two application keys, consumer and secret. Put the two keys into the appropriate boxes.
